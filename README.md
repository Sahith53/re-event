# re:Event - Event Management Platform

A comprehensive event management platform that streamlines event planning, ticketing, and attendee management.

## Project Overview

re:Event is a modern event management platform that helps hosts create, manage, and enhance their events with features like real-time collaboration, smart resource allocation, and automated scheduling.

## Features

- **Streamlined Planning:** Intuitive tools for event planning and management
- **Secure Attendee Access:** Secure RSVP and check-in system
- **Event Discovery:** Find and explore local events
- **Real-time Collaboration:** Work with other hosts in real-time
- **Customizable Event Pages:** Create unique event pages with diverse layouts
- **Registration Management:** Approve or deny event registrations
- **Multiple Event Types:** Support for online, mid-scale, and large-scale events
- **Payment Integration:** Support for both free and paid events
- **Email Notifications:** Automated email notifications for attendees
- **Location Services:** Integration with maps for precise event locations
- **Analytics:** Detailed insights and analytics for each event

## Tech Stack

- Frontend: React, Vite, TailwindCSS
- Backend: Node.js, Express, MongoDB
- Authentication: JWT, Session-based
- Additional Tools: Chart.js, QR Code Generation, Email Services

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- MongoDB
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   # Frontend
   cd frontend
   npm install

   # Backend
   cd backend
   npm install
   ```

3. Set up environment variables
4. Start the development servers:
   ```bash
   # Frontend
   npm run dev

   # Backend
   npm start
   ```
